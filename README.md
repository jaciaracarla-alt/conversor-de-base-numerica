# Conversor de Bases Numéricas (versão fofinha)

**Descrição**  
Aplicação simples em HTML, CSS e JavaScript para converter números entre bases: binário, octal, decimal e hexadecimal. Tem tema “fofinho” (kawaii) com inputs estilizados e botões 3D.

---

## Arquivos principais
- `index.html` — página principal (interface).  
- `style.css` — estilos (tema fofo, botões 3D).  
- `script.js` — lógica de conversão.


---

## Funcionalidades
- Entrada de número em base escolhida (2, 8, 10, 16).  
- Conversão automática para as demais bases (decimal, binário, octal, hexadecimal).  
- Validação simples de entrada (alerta se inválido).  
- Visual agradável, responsivo e com botões “fofinhos”.

---

## Como executar (local)
1. Clone ou copie os arquivos para uma pasta local.  
2. Abra o arquivo `index.html` no navegador (duplo clique ou `File -> Open` no browser).  
3. Digite o número, escolha a base e clique em **✨ Converter ✨**.

---

## Exemplos de uso
- Entrada: `1010` com base `2` → Decimal: `10`, Hex: `A`.  
- Entrada: `255` com base `10` → Binário: `11111111`, Hex: `FF`.
